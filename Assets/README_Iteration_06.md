# Loop Puzzle — Итерация 6: Уровни, прогрессия, сохранения, звёзды

## Что нового в этой итерации

- **25 уникальных уровней** с явной прогрессией по 5 мирам (по 5 уровней в мире). Каждый уровень — визуально другая форма петли, растущее поле и число элементов.
- **Экран выбора уровней** — вертикально-прокручиваемый список, сгруппированный по мирам (заголовок мира + сетка кнопок 5 в ряд, всё скроллится).
- **Система прогрессии и сохранений** (PlayerPrefs): открытие следующего уровня после прохождения, лучший результат по ходам, заработанные звёзды.
- **Звёзды (0–3)** — реальный подсчёт и показ в попапе победы с анимацией появления.
- **Кнопка PLAY** в меню теперь продолжает с последнего открытого уровня; **LEVELS** открывает экран выбора; в попапе победы есть **NEXT** (если есть следующий уровень) и **LEVELS** (назад к выбору).
- **Двойная валидация уровней** — все уровни проверены при генерации (Python) и проверяются прямо в Unity при импорте; есть отдельная меню-команда валидатора.

## Как обеспечена 100% проходимость

Уровни не написаны «на глаз». Они сгенерированы и проверены программно, той же логикой петель, что и в игре:

1. **Доказательство построением** (все 25 уровней): для каждого уровня известно эталонное решение (петля собрана и проверена). Стартовое состояние получено из решения через ровно те ходы, что доступны игроку (поворот на +90° и перетаскивание смещённых элементов на пустые клетки). Затем «обратная сборка» из старта снова проверяется на победу. Если из старта можно дойти до победы доступными ходами — уровень проходим.
2. **Независимый перебор** (уровни 1–20): отдельный решатель ПОЛНЫМ перебором поворотов (игнорируя эталонное решение) подтвердил, что выигрышная конфигурация существует. Уровни 21–25 — это чистые прямоугольные петли, построенные тем же проверенным генератором, и покрыты доказательством построением (перебор 4^16+ слишком велик, но форма гарантированно корректна).
3. Проверки на каждом уровне: решение даёт нужное число замкнутых петель; старт ещё НЕ решён; типы элементов сохраняются; смещённые элементы совпадают по типу с целевыми клетками; зафиксированные элементы уже на правильном повороте; всё в границах поля; ровно один маркер старта.

Результат генератора: **ALL LEVELS VALID & UNIQUE**.

## Валидатор прямо в Unity

Два пути:

- При импорте (`Tools → Loop Puzzle → Setup Iteration 6`) каждый уровень прогоняется через `LevelValidator`. Если хоть один не проходит — импорт прерывается с ошибкой в консоли, база не собирается.
- В любой момент: `Tools → Loop Puzzle → Validate All Levels` — проверяет все уровни в базе и проверяет уникальность раскладок, печатает результат в консоль.

`LevelValidator` повторяет ту же модель соединений и подсчёта петель, что и игровой `LoopValidator`, но работает с ассетом `LevelData` без сцены.

## Условие звёзд

- 1 звезда — уровень пройден.
- +1 звезда — без использования подсказок (подсказки появятся в итерации 7; поле `hintUsed` уже заведено).
- +1 звезда — пройдено за число ходов ≤ Par.

## Установка

1. В проекте должны быть импортированы **DOTween** (бесплатный) и **TMP Essentials** (как в прошлых итерациях).
2. Скопируй папку `Assets/LoopPuzzle/` в проект поверх предыдущей итерации.
3. Запусти `Tools → Loop Puzzle → Setup Iteration 6`.

Что сделает Setup:
- импортирует `levels_export.json` и создаст 25 ассетов `LevelData` в `Assets/LoopPuzzle/Levels/`;
- провалидирует каждый уровень (прервётся при ошибке);
- соберёт `LevelDatabase` в `Assets/LoopPuzzle/Data/`;
- добавит `ProgressManager` в сцену MainMenu;
- создаст сцену `LevelSelect` (скролл-список по мирам);
- допишет в попап победы строку звёзд и кнопку LEVELS, прокинет базу в `LevelController`;
- добавит 3 сцены в Build Settings (MainMenu, LevelSelect, Game).

## Как проверить

- Меню → PLAY: грузит последний открытый уровень. На старте открыт только уровень 1.
- Меню → LEVELS: скролл-список; закрытые уровни с замком, открытые — с числом звёзд.
- Пройди уровень 1 (поворачивай элементы тапом). При победе: анимация петли, попап со звёздами, открывается уровень 2.
- NEXT в попапе ведёт на следующий уровень, LEVELS — на экран выбора.
- `Tools → Loop Puzzle → Validate All Levels` — проверка всех уровней в консоли.

## Заметки

- Анимация потока на уровнях Мира 4 (две петли) прослеживает петлю, содержащую маркер старта — это нормально визуально.
- Частицы и звуки пока временные (заглушки), как и раньше.
- Кнопка Settings пока выводит сообщение в консоль (полноценные настройки — итерация 7).
- Звёзды и замок в списке нарисованы символами (`*`, `X`) как временные глифы; в полировке (итерация 9) заменим на спрайты.
