# Loop Puzzle — Итерация 7: Undo / Restart / Hint + Настройки

## Что нового

- **Undo (отмена хода)** — кнопка в нижней панели игры. Откатывает последний ход (поворот или перетаскивание) с обратной анимацией, уменьшает счётчик ходов. Доступна только когда есть что отменять. Undo — базовая механика, на звёзды НЕ влияет.
- **Restart (перезапуск)** — кнопка в панели. Возвращает уровень в стартовое состояние, обнуляет ходы и историю.
- **Hint (подсказка, Вариант В)** — подсвечивает один проблемный элемент и показывает стрелку направления поворота, но НЕ крутит за игрока. Игрок сам выполняет ход. Использование подсказки выставляет флаг и убирает звезду «без подсказок».
- **Настройки (Settings)** — полноценный попап вместо заглушки: тумблеры Звук / Музыка / Вибрация и кнопка сброса прогресса (с подтверждением в два тапа). Доступен из главного меню.
- **Editor-команды:**
  - `Tools → Loop Puzzle → Unlock All Levels` — разблокирует все уровни.
  - `Tools → Loop Puzzle → Lock All Levels (reset to 1)` — заблокировать обратно и сбросить прогресс.

## Как работает подсказка (и почему она всегда верна)

В каждый уровень теперь записано проверенное решение (правильный тип и поворот элемента для каждой клетки финальной петли). Подсказка сравнивает текущее состояние поля с решением:

1. Ищет элемент, который стоит на своей клетке, но повёрнут неправильно → подсвечивает его и рисует стрелку поворота.
2. Если все на местах по повороту, но есть смещённый элемент (Миры 2+) → подсвечивает его и целевую пустую клетку.

Решения экспортированы из того же Python-генератора, что проверял проходимость в итерации 6. Дополнительно проверено: применение всех «правильных» поворотов из решения к стартовому полю даёт победу на всех 25 уровнях. То есть стрелки подсказки ведут ровно к решённому состоянию.

## Как работает Undo

История ходов — стек. Каждый ход пишется при совершении:
- поворот → запоминается клетка; отмена = поворот назад на 90° с обратной анимацией;
- перетаскивание → запоминается «откуда» и «куда»; отмена = вернуть элемент назад.

Счётчик ходов при отмене уменьшается. История очищается при загрузке/перезапуске уровня и блокируется после победы.

## Установка

1. В проекте уже должны быть DOTween и TMP Essentials, а также выполнены Setup итераций 1–6 (нужны сцены, база уровней и ассеты уровней).
2. Скопируй папку `Assets/LoopPuzzle/` в проект поверх итерации 6.
3. Запусти `Tools → Loop Puzzle → Setup Iteration 7`.

Что сделает Setup:
- допишет данные решения в 25 существующих ассетов `LevelData` (re-import из обновлённого `levels_export.json`);
- добавит в игровую сцену объект `HintSystem` и нижнюю панель с кнопками UNDO / RESTART / HINT, прокинет ссылки в `HUDController` и `LevelController`;
- построит попап настроек в сцене MainMenu и подключит к `SoundManager` / `HapticManager` / `ProgressManager`.

## Как проверить

- Зайди в уровень. Внизу — три кнопки: UNDO (серая/неактивная пока нет ходов), RESTART, HINT.
- Покрути несколько элементов → UNDO активируется, откатывает по одному ходу, счётчик уменьшается.
- Нажми HINT → один неправильный элемент подсветится + стрелка поворота. Сам доверни его.
- Пройди уровень с подсказкой → в попапе будет максимум 2 звезды (потеряна «без подсказок»).
- RESTART → уровень сбрасывается в начальное состояние.
- Меню → SETTINGS: переключай Звук/Музыка/Вибрация; Reset Progress (два тапа) сбрасывает прогресс.
- `Tools → Loop Puzzle → Unlock All Levels` → все уровни открыты в списке.

## Заметки

- Стрелка подсказки и иконки кнопок нарисованы кодом (спрайт стрелки генерируется в `PieceSpriteFactory`); в полировке (итерация 9) можно заменить на художественные.
- Подсказка показывает один шаг за раз — нажми снова для следующего проблемного элемента.
- Анимация частиц/звуки пока заглушки, как и раньше.
- Кнопки настроек — простые тумблеры (цвет + текст On/Off); полноценные свитчи/иконки — в полировке.
