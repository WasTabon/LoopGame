# README — Iteration 01: Foundation + Main Menu

## 1. Что было добавлено
Фундамент проекта: менеджеры SoundManager, HapticManager, TransitionManager (DontDestroyOnLoad синглтоны), главное меню с кнопками Play / Levels / Settings и анимациями появления, плавный fade-переход между сценами. Создаётся точка входа и пустая игровая сцена.

## 2. Что изменилось с прошлой итерации
Первая итерация — изменений нет.

## 3. Установка (ОБЯЗАТЕЛЬНО прочитать перед запуском)

### Шаг 1. DOTween
Проект использует **DOTween (free)**. Установи его ПЕРЕД запуском editor-скрипта:
- Asset Store → DOTween (free) → Import
- После импорта откроется окно настройки DOTween → нажми **Setup DOTween...** → **Apply**
- Без этого скрипты не скомпилируются (ошибки на `using DG.Tweening;`)

### Шаг 2. TextMeshPro
При первом использовании TMP Unity попросит импортировать **TMP Essentials**:
- Window → TextMeshPro → Import TMP Essential Resources

### Шаг 3. Копирование файлов
Скопируй папку `Assets/LoopPuzzle/` из архива в свой проект (в папку `Assets/`).

## 4. Editor скрипты: что запустить
После того как проект скомпилировался без ошибок:

- В верхнем меню Unity: **Tools → Loop Puzzle → Setup Iteration 1**

Скрипт автоматически:
- создаст сцены MainMenu и Game в `Assets/LoopPuzzle/Scenes/`
- расставит камеры, EventSystem, менеджеры
- сгенерирует весь UI меню и игровой сцены, прокинет все ссылки
- добавит обе сцены в Build Settings
- откроет сцену MainMenu

## 5. Как тестировать
1. Открой сцену `Assets/LoopPuzzle/Scenes/MainMenu.unity` (скрипт открывает её сам).
2. Нажми **Play** в Unity.
3. Жми кнопки PLAY / LEVELS / BACK, проверь переходы и звуки.

## 6. Ожидаемый результат
- При старте экран плавно проявляется из чёрного.
- Заголовок и кнопки появляются друг за другом с эффектом «отскока» (Ease.OutBack).
- При нажатии кнопка слегка сжимается, играет звук-клик, на устройстве — вибрация.
- PLAY и LEVELS уводят на игровую сцену через fade; BACK возвращает в меню.
- SETTINGS пока пишет лог в консоль (попап будет в итерации 7).
- Игровая сцена — пустой заглушечный экран с надписью GAME SCENE и кнопкой BACK.

## 7. Известные ограничения текущей итерации
- Звуки — сгенерированные кодом «бипы»-плейсхолдеры (потом заменятся реальными).
- Музыки пока нет (MusicSource создан, но без клипа).
- Вибрация в редакторе не ощущается — только на реальном устройстве (iOS/Android).
- Settings и Levels пока не функциональны (Levels временно ведёт на Game).
- Игровая сцена пустая — поле появится в итерации 2.
