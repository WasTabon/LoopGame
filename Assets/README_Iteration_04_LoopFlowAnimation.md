# README — Iteration 04: Loop Flow Animation + Win Polish

## 1. Что было добавлено
Главный game feel победы. После сборки loop проигрывается секвенция: волна подсветки по элементам → светящаяся точка со шлейфом пробегает контур (~2 круга) → вспышка частиц в центре + тряска камеры + сильная вибрация → затем попап победы. Скорость потока зависит от длины маршрута (большие loop не тормозят).

## 2. Что изменилось с прошлой итерации
- `LevelController` теперь при победе запускает анимацию потока и показывает попап ТОЛЬКО после её завершения (раньше была просто пауза 0.4 сек).
- `PathPiece` получил подсветку (HighlightPulse), сброс визуала (ResetVisual) и безопасную очистку твинов при уничтожении.
- `PieceSpriteFactory` рисует светящуюся точку (радиальный glow).
- `SoundManager` получил звук пробега энергии (восходящий свип).
- Добавлен `CameraShake` на Main Camera.
- Всё из итераций 1–3 на месте.

## 3. Editor скрипты: что запустить
ВАЖНО: итерации 1–3 уже должны быть установлены.

1. Скопируй папку `Assets/LoopPuzzle/` из архива поверх своего проекта.
2. Дождись компиляции без ошибок.
3. В меню Unity: **Tools → Loop Puzzle → Setup Iteration 4**

Скрипт добавит CameraShake на камеру, создаст объект LoopFlowAnimator (с генерируемой кодом точкой-шлейфом и системой частиц), прокинет ссылки из LevelController в аниматор. Сохранит сцену.

## 4. Как тестировать
1. Открой сцену `Assets/LoopPuzzle/Scenes/Game.unity`.
2. Нажми **Play**.
3. Собери замкнутый loop (поверни углы правильно).
4. Наблюдай победную секвенцию: вспышки по элементам → бегущая светящаяся точка по контуру → взрыв частиц + тряска → попап.
5. Проверь на Level_02 (5x5) — поток длиннее, но скорость та же (масштабируется).
6. Нажми Restart/Next и собери ещё раз — анимация должна корректно проигрываться повторно.

## 5. Ожидаемый результат
- Элементы loop по очереди вспыхивают тёплым светом с punch-масштабом.
- Светящаяся точка с оранжевым шлейфом плавно бежит по замкнутому контуру ~2 круга, начиная со стартовой клетки.
- Играет звук-свип во время пробега.
- В конце — вспышка частиц в геометрическом центре loop, короткая тряска камеры, сильная вибрация.
- Только после всего этого всплывает попап победы.

## 6. Известные ограничения текущей итерации
- Частицы/точка — простые кодовые плейсхолдеры (спрайт-glow + Sprites/Default material), потом можно заменить на красивые ассеты.
- Звёзды в попапе всё ещё плейсхолдер — итерация 6.
- NEXT по-прежнему перезагружает текущий уровень — итерация 6.
- Перемещения элементов (drag) пока нет — итерация 5.
- Тайминги подобраны на глаз; при желании потом подправим в полировке (итерация 9).

## 7. Что будет в следующей итерации
Free drag — перемещение элементов в любую пустую клетку (ядро Мира 2): зажал, перетащил, отпустил, snap или возврат.
